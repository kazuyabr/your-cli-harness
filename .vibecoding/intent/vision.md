# Vision

## Your CLI Harness — Visão Geral

O **Your CLI Harness** é uma plataforma de geração de CLIs de IA white-label. Ele permite que qualquer cliente crie sua própria ferramenta de linha de comando com identidade visual, agentes, modos e integrações personalizadas — sem reescrever a lógica de interação com LLMs.

## Propósito

Separar o **núcleo funcional** da **camada de marca**, permitindo criar CLIs com aparência, nome e comunicação específicas a partir de uma base reutilizável.

## Valores Centrais

1. **Simplicidade** — Começar simples, adicionar complexidade apenas quando necessário
2. **Transparência** — Mostrar o que o agente está fazendo e planejando
3. **Eficiência** — Tokens são caros; cada linha de contexto deve justificar seu custo
4. **Modularidade** — Core isolado, clientes plugáveis, extensões via MCP/Skills
5. **Padrões Abertos** — Compatível com ecossistema Anthropic (Skills, MCP, CLAUDE.md)

## Inspiração

- **CodeWhale** — Posicionamento e abordagem de CLI personalizável
- **OpenCode** — Arquitetura de agentes e modos
- **Claude Code** — Sistema de memória, skills, subagentes, MCP
- **Anthropic Engineering** — Padrões de agentes eficazes (building effective agents)

## Cliente Inicial

**Jogatinando** — Primeira CLI gerada a partir do harness, servindo como prova de caso e referência de implementação.
